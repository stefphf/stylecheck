# StyleCheck — Проверка документов по ГОСТ

Веб-сервис автоматической проверки DOCX-документов на соответствие
стандартам оформления (ГОСТ 7.32-2017, ГОСТ Р 7.0.11-2011 и др.)

---

## Запуск

Нужно **два открытых терминала**.

### Терминал 1 — Бэкенд (API)

```bash
cd backend
pip install -r requirements.txt
python app.py
```

Бэкенд запустится на http://localhost:8000  
Документация API: http://localhost:8000/docs

### Терминал 2 — Фронтенд (React)

```bash
cd frontend-src
npm install
npm run dev
```

Откроется на http://localhost:5173

## Что проверяется в документе

- **Структура** — наличие обязательных разделов, иерархия заголовков
- **Терминология** — запрещённые слова с морфологическим анализом
- **Оформление** — двойные пробелы, точки в заголовках, длина абзацев
- **ГОСТ** — поля страницы, шрифт Times New Roman 14pt, интервал 1.5×, отступ 1.25 см

Каждое нарушение содержит: описание, номер страницы, номер абзаца и цитату из текста.
